# 4 多元线性回归

## 4.1 多个自变量

在多元线性回归中有不止一个自变量：

$h_\theta{(x)}=\theta_0+\theta_1{x_1}+\theta_2{x_2}+\theta_3{x_3}+\theta_4{x_4}$

在这里，可以认为有自变量$x_0===1$

## 4.2 多元的梯度下降

同一元情况下的$\theta_0$和$\theta_1$，但是在这里有4个$\theta$，分别对4个$\theta$求偏导，然后根据偏导同时更新4个$\theta$的参数值，直到损失函数$J(\theta_0, \theta_1, \theta_2, \theta_3, \theta_4)$收敛。

## 4.3 实操技巧1：特征归一化(Feature Scaling)

使用公式$\frac{x_i-\mu_i}{\sigma}$把每个连续特征归一化到 $[-1,1]$ 空间，可以提升模型效果。

## 4.4 实操技巧2：调整学习率$\alpha$

通过监控训练过程中损失函数$J(\theta)$的值，来调整学习率。如果$\alpha$太大，则$J(\theta)$可能不会收敛或者出现跳跃，如果$\alpha$太小，则$J(\theta)$收敛过慢。

## 4.5 多项式回归

某些自变量可能与因变量直接不是线性关系，可以考虑使用自变量的平方项或者立方项等当作x，例如：

$h_\theta{(x)}\\=\theta_0+\theta_1{x_1}+\theta_2{x_2}+\theta_3{x_3}+\theta_4{x_4}\\=\theta_0+\theta_1{(size)}+\theta_2{(size)^2}+\theta_3{(size)^3}+\theta_4{\sqrt{(size)}}$

## 4.6 标准方程

在**均方误差损失函数**下，通过**二次方程求根公式**，$\theta$参数的解可以用标准方程得到：
$$\theta=(X^TX)^{-1}X^Ty$$
例如，当有m个样本，n个特征时：$X$的维度为$m\times{n}$，$Y$的维度为$m\times{1}$，最终输出维度为$n\times{1}$。

以下是**梯度下降法和标准方程的对比**：

| 梯度下降                  | 标准方程                                |
| ------------------------- | --------------------------------------- |
| 需要选择学习率$\alpha$    | 不需要选择学习率$\alpha$                |
| 需要迭代很多次            | 不需要迭代                              |
| 特征维度$n$很大时依然好用 | 需要计算$(X^TX)^{-1}$，复杂度为$O(n^3)$ |
|                           | 当特征维度$n$很大时，计算缓慢           |

## 4.7 标准方程中的$(X^TX)^{-1}$不可逆

标准方程中的$(X^TX)^{-1}$项是有可能不可逆的，出现不可逆的可能情况有：

- **特征冗余**，当两个特征线性相关时
- **特征数量过多(m≤n)**，此时可以**删除一些特征**，或者使用**正则化**。

**在Octave/Matlab中，使用pinv函数可以在矩阵不可逆时依然得到逆矩阵。*
