# XGBoost

XGBoost是GBDT的一个变种，本质上是很多CART(Classification and Regression Tree)模型的集成，而CART本质上是一个决策树模型。

## 决策树

决策树可以简单理解为if-then规则的集合，也可以认为是定义在特征空间与类空间上的条件概率分布。

<树图片> 

优点：可读性好，分类速度快。

|          | ID3       | C4.5       | CART                          |
| -------- | --------- | ---------- | ----------------------------- |
| 学习目标 | 分类      | 分类       | 分类、回归                    |
| 模型     | 二叉树    | 二叉树     | 二叉树                        |
| 损失     | 信息增益  | 信息增益率 | 平方和损失/Gini比率/LogitLoss |
| 优化算法 | 排序+遍历 | 排序+遍历  | 排序+遍历                     |
