1. 强化学习概念

强化学习是一个 Agent 与 Environment 相互交互的过程

​ state s∈S

​ action a∈A

​ ——————

​ | ↓

Agent Environment

​ ↑ |

​ ——————

​ Get reward r

​ New state s'∈S

其中重要的概念有如下：

  • States状态空间：包含所有可能的状态

  • Actions动作空间：包含所有可能的动作

  • Rewards奖励：R（s, a）在不同的状态采取相同或不同的动作都有对应的一个奖励

  • Values期望值：总奖励

  • Policy：策略，代表在某个状态s的最佳动作action

  • Episode：通过一系列动作经过一系列状态后，获得的一个序列，例如：S1 A1 R2 S2 A2 R3 S3 …… ST（终止）

2. Bellman等式 - 奖励函数的计算方法

Bellman equation是求解每个状态s的V值的过程，只要求解出每个状态的V值后，我们就可以通过每次都选择最大V值来获得最优策略。

Bellman equation (1):

V(s)=E[Gt∣St=s]=E[Rt+1+γRt+2+γ2Rt+3+⋯∣St=s]=E[Rt+1+γ(Rt+2+γRt+3+⋯ )∣St=s]=E[Rt+1+γGt+1∣St=s]=E[Rt+1+γV(St+1)∣St=s]\begin{aligned} V(s)&=E[G_t\mid S_t=s] \\ &=E[R_{t+1}+\gamma R_{t+2}+\gamma^{2}R_{t+3}+\cdots \mid S_t=s] \\ &=E[R_{t+1}+\gamma(R_{t+2}+\gamma R_{t+3}+\cdots)\mid S_t=s] \\ &=E[R_{t+1}+\gamma G_{t+1}\mid S_t=s] \\ &=E[R_{t+1}+\gamma V(S_{t+1})\mid S_t=s] \end{aligned} V(s)​=E[Gt​∣St​=s]=E[Rt+1​+γRt+2​+γ2Rt+3​+⋯∣St​=s]=E[Rt+1​+γ(Rt+2​+γRt+3​+⋯)∣St​=s]=E[Rt+1​+γGt+1​∣St​=s]=E[Rt+1​+γV(St+1​)∣St​=s]​

可以看到，这是一个递归，所以要求解V(s)，只需要从末端反推就可以。

Bellman equation (2):

Q(s,a)=E[Rt+1+γV(St+1)∣St=s,At=a]=E[Rt+1+γEa=π[Q(st+1,a)]∣St=s,At=a]\begin{aligned} Q(s,a)&=E[R_{t+1}+\gamma V(S_{t+1})\mid S_t=s,A_t=a] \\ &=E[R_{t+1}+\gamma E_{a=\pi}[Q(s_{t+1},a)]\mid S_t=s,A_t=a] \end{aligned} Q(s,a)​=E[Rt+1​+γV(St+1​)∣St​=s,At​=a]=E[Rt+1​+γEa=π​[Q(st+1​,a)]∣St​=s,At​=a]​

通过求解max(Qa_1, Qa_2, Qa_3,...)来获得当前步最佳策略action。

3. Dynamic Programming - 奖励函数的求解方法

Value iteration 值迭代

具体做法是：先对所有状态初始化一个值，然后使用 Bellman equation

V(St)=r+∑PP(St+1∣St)V(St+1)V(S_t)=r+\sum_{P}P(S_{t+1}\mid S_t)V(S_{t+1}) V(St​)=r+P∑​P(St+1​∣St​)V(St+1​)

进行值的更新，然后不断迭代，直到收敛 ∣Wt+1−Wt∣≤δ|W_{t+1}-W_t|\leq \delta∣Wt+1​−Wt​∣≤δ：

其中有两种做法，一种是一次计算，同时更新，另一种是将更新后的值作为下一个格子的输入值进行更新。这两种做法都是可取的。

Policy iteration 策略迭代

具体做法是：先对所有状态初始化一个随机策略，然后使用第二种Bellman equation，进行最佳策略的更新，然后不断迭代，直到收敛。

4. Monte-Carlo Method - 奖励函数的求解方法2

正常情况下一般来讲，各状态之间的Transition Probability是未知的。

蒙特卡洛是用随机模拟的方式，来估算每一个状态的V值的方法。

S0→ra0S1→ra1S2→ra2⋯→rat−1ST  (1)S_0 \xrightarrow[r]{a_0} S_1 \xrightarrow[r]{a_1} S_2 \xrightarrow[r]{a_2} \cdots \xrightarrow[r]{a_{t-1}} S_T \; \text{(1)} S0​a0​r​S1​a1​r​S2​a2​r​⋯at−1​r​ST​(1)

S0′→ra0S1′→ra1S2′→ra2⋯→rat−1ST′  (2)S_0' \xrightarrow[r]{a_0} S_1' \xrightarrow[r]{a_1} S_2' \xrightarrow[r]{a_2} \cdots \xrightarrow[r]{a_{t-1}} S_T' \; \text{(2)} S0′​a0​r​S1′​a1​r​S2′​a2​r​⋯at−1​r​ST′​(2)

……

通过不同的模拟，我们可以得到不同的路线，而每次模拟都可以算出路线上每个状态点的 GtG_tGt​ 值。

而真 GtG_tGt​ 值的计算有 2 种方法：

① First Visited - 每个状态在每条路径只计算第一次

V(S1)=Gt1+Gt2+Gt3+⋯+GtnnV(S_1)=\frac{G_t^1+G_t^2+G_t^3+\cdots+G_t^n}{n} V(S1​)=nGt1​+Gt2​+Gt3​+⋯+Gtn​​

② Every Visited - 每个状态碰见几次就是几次

V(S1)=Gt1(S1)+Gt2(S1)+Gt3(S1)+⋯+Gtn(S1)nV(S_1)=\frac{G_t^1(S_1)+G_t^2(S_1)+G_t^3(S_1)+\cdots+G_t^n(S_1)}{n} V(S1​)=nGt1​(S1​)+Gt2​(S1​)+Gt3​(S1​)+⋯+Gtn​(S1​)​

总体来讲，蒙特卡洛方法bias低（为0），variance高（由于随机采样）

DP法bias高（需要估计概率值），variance低（不需随机采样，直接计算）

6. Q-learning and SARSA

Q-learning: 计算出不同的Q后，Value取值按照max()取

SARSA (State-Action-Reward-State-Action): 计算出不同的Q后，Value取值按照policy取

7. Function Approximation (Deep Q-Network)

Deep Q-Network的思路是使用神经网络来拟合Q函数：f(S1) → Q值

该神经网络的输入就是State，输出就是Q-Value

（例如，Atari游戏中，输入的就是图片像素，使用CNN提取特征）
