7.1 过拟合问题

过拟合：如果特征过多，模型可能对训练集有很好的效果，但泛化能力很差

解决过拟合的方法：

  1. 减少特征数量
    • 手工选取重要特征
    • 特征选择算法（后面会介绍）
  2. 正则化
    • 保留所有特征，但是对参数 θj\theta_jθ​j​​ 的量级加以限制
