4 多元线性回归

4.1 多个自变量

在多元线性回归中有不止一个自变量：

hθ(x)=θ0+θ1x1+θ2x2+θ3x3+θ4x4h_\theta{(x)}=\theta_0+\theta_1{x_1}+\theta_2{x_2}+\theta_3{x_3}+\theta_4{x_4}hθ​(x)=θ0​+θ1​x1​+θ2​x2​+θ3​x3​+θ4​x4​

在这里，可以认为有自变量x0===1x_0===1x0​===1

4.2 多元的梯度下降

同一元情况下的θ0\theta_0θ0​和θ1\theta_1θ1​，但是在这里有4个θ\thetaθ，分别对4个θ\thetaθ求偏导，然后根据偏导同时更新4个θ\thetaθ的参数值，直到损失函数J(θ0,θ1,θ2,θ3,θ4)J(\theta_0, \theta_1, \theta_2, \theta_3, \theta_4)J(θ0​,θ1​,θ2​,θ3​,θ4​)收敛。

4.3 实操技巧1：特征归一化(Feature Scaling)

使用公式xi−μiσ\frac{x_i-\mu_i}{\sigma}σxi​−μi​​把每个连续特征归一化到 [−1,1][-1,1][−1,1] 空间，可以提升模型效果。

4.4 实操技巧2：调整学习率α\alphaα

通过监控训练过程中损失函数J(θ)J(\theta)J(θ)的值，来调整学习率。如果α\alphaα太大，则J(θ)J(\theta)J(θ)可能不会收敛或者出现跳跃，如果α\alphaα太小，则J(θ)J(\theta)J(θ)收敛过慢。

4.5 多项式回归

某些自变量可能与因变量直接不是线性关系，可以考虑使用自变量的平方项或者立方项等当作x，例如：

hθ(x)=θ0+θ1x1+θ2x2+θ3x3+θ4x4=θ0+θ1(size)+θ2(size)2+θ3(size)3+θ4(size)h_\theta{(x)}\\=\theta_0+\theta_1{x_1}+\theta_2{x_2}+\theta_3{x_3}+\theta_4{x_4}\\=\theta_0+\theta_1{(size)}+\theta_2{(size)^2}+\theta_3{(size)^3}+\theta_4{\sqrt{(size)}}hθ​(x)=θ0​+θ1​x1​+θ2​x2​+θ3​x3​+θ4​x4​=θ0​+θ1​(size)+θ2​(size)2+θ3​(size)3+θ4​(size)​

4.6 标准方程

在均方误差损失函数下，通过二次方程求根公式，θ\thetaθ参数的解可以用标准方程得到：

θ=(XTX)−1XTy\theta=(X^TX)^{-1}X^Ty θ=(XTX)−1XTy

例如，当有m个样本，n个特征时：XXX的维度为m×nm\times{n}m×n，YYY的维度为m×1m\times{1}m×1，最终输出维度为n×1n\times{1}n×1。

以下是梯度下降法和标准方程的对比：

梯度下降标准方程
需要选择学习率α\alphaα不需要选择学习率α\alphaα
需要迭代很多次不需要迭代
特征维度nnn很大时依然好用需要计算(XTX)−1(X^TX)^{-1}(XTX)−1，复杂度为O(n3)O(n^3)O(n3)
当特征维度nnn很大时，计算缓慢

4.7 标准方程中的(XTX)−1(X^TX)^{-1}(XTX)−1不可逆

标准方程中的(XTX)−1(X^TX)^{-1}(XTX)−1项是有可能不可逆的，出现不可逆的可能情况有：

  • 特征冗余，当两个特征线性相关时
  • 特征数量过多(m≤n)，此时可以删除一些特征，或者使用正则化。

*在Octave/Matlab中，使用pinv函数可以在矩阵不可逆时依然得到逆矩阵。
