3 线性代数复习（Optional）

3.1 矩阵和向量

iii行，jjj列的矩阵可以写作：Ri×j\R^{i\times{j}}Ri×j

3.2 加法和数乘

加法：维度完全一样的两个矩阵，按元素相加，输出相同维度的一个矩阵

数乘：对矩阵中每个元素全部乘以或除以某个实数

3.3 矩阵与向量相乘

Ri×j×Rj×1=Ri×1\R^{i\times{j}}\times\R^{j\times{1}}=\R^{i\times{1}}Ri×j×Rj×1=Ri×1

3.4 矩阵与矩阵相乘

Ri×j×Rj×k=Ri×k\R^{i\times{j}}\times\R^{j\times{k}}=\R^{i\times{k}}Ri×j×Rj×k=Ri×k

3.5 矩阵乘法的属性

3.6 逆矩阵与转置

逆矩阵：如果矩阵Am×mA^{m\times{m}}Am×m有逆矩阵A−1 A^{-1}A−1，则它们满足AA−1=A−1A=IAA^{-1}=A^{-1}A=IAA−1=A−1A=I。

不是所有矩阵都有逆矩阵，没有逆矩阵的矩阵被称为奇异矩阵(singular)或退化矩阵(degenerate)。
